# Directory Structure Rules

- `/notes`: Contains markdown summaries of exam topics (e.g., `kms-key-types.md`).
- `/labs`: Contains reproducible scenarios. Each lab must have a `README.md` with a "Challenge" section and a `solution/` folder.
- `/diagrams`: Contains Mermaid code for security flows (e.g., OIDC Federation flow).
- `/exercises`: Contains some exercises baded on the notes
- `/examples`: Contains some examples for specific resources should include a index.md where all examples
are referenced
